# Projekt ćwiczeniowy PHP

Krótki projekt stworzony w ramach ćwiczeń. Ma na celu prezentację podstawowej aplikacji webowej w PHP uruchamianej lokalnie na XAMPP.

Technologie:
- PHP
- MySQL (MariaDB)
- HTML/CSS, JavaScript
- XAMPP

Szybkie uruchomienie:
1. Skopiuj katalog projektu do folderu htdocs w instalacji XAMPP.
2. Uruchom Apache i MySQL w panelu XAMPP.
3. Otwórz przeglądarkę pod adresem: http://localhost/ĆwiczeniaPHP/projekt

Struktura (przykładowo):
- public/ (plik wejściowy, zasoby)
- src/ (kod PHP)
- sql/ (skrypty tworzenia bazy danych)

Opis funkcjonalny:
- Proste operacje CRUD na przykładowych danych, autoryzacja podstawowa, formularze i walidacja po stronie serwera.

Uwagi:
- Dostosuj wersje PHP i ustawienia bazy danych do swojej instalacji XAMPP.
